# Зображення сайту YULIIA&MAX — структура

Папки відповідають секціям лендингу (порядок зверху вниз).

## 01-hero/ — Hero (перший екран)
- `hero-couple.png` — фото пари YULIIA & MAX на фоні першого екрану (full-bleed background).

## 02-pro-nas/ — Про нас
- `about-couple.png` — фото пари з телефонами на фоні секції "Про нас" (mobile) / у фото-карті справа (desktop).

## 02-nashi-rezultaty/ — Наші результати (поляроїди-події)
Дві пари поляроїд-карток у секції "Цифри, які говорять самі":
- `01-polaroid-event-1.webp` — перша пара, ліве фото (rotate -6°)
- `02-polaroid-event-2.webp` — перша пара, праве фото (rotate +5°)
- `03-polaroid-event-3.webp` — друга пара, ліве фото
- `04-polaroid-event-4.webp` — друга пара, праве фото

## 03-pro-brand/ — Про бренд
- `01-logo.webp` — логотип бренду (квадратна карта праворуч від заголовка)
- `02-polaroid-product-1.webp` — продуктова поляроїд-картка №1 (центр секції)
- `03-polaroid-product-2.webp` — продуктова поляроїд-картка №2 (центр секції)

## 05-vidguki/ — Відгуки нашої команди (слайдер скріншотів)
Слайдер зі скріншотами повідомлень / відгуків:
- `01-testimonial-screenshot.webp` — слайд 1
- `02-testimonial-screenshot.webp` — слайд 2
- `03-testimonial-screenshot.webp` — слайд 3

**Не заповнено (порожні слоти на сайті):**
- Продукція бренду → `banner-1`..`banner-5` (слайдер банерів категорій)
- Відгуки → `tm-4`, `tm-5`, `tm-6` (додаткові слайди)

Ці слоти на сайті залишаються drag-and-drop placeholder'ами — додайте файли коли будуть готові.
